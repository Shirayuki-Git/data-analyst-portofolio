Sales Performance Dashboard

## Deskripsi
Proyek ini bertujuan untuk menganalisis kinerja penjualan berdasarkan data transaksi
guna melihat tren penjualan, performa sales, dan kontribusi wilayah.

## Dataset
- Jenis data: Data penjualan
- Isi data: tanggal transaksi, sales, wilayah, unit price, total amount
- Sumber data: dataset latihan / data dummy

## KPI yang Dianalisis
- Total Sales
- Total Customer
- Unit Price

## Analisis Utama
- Analisis tren penjualan dari waktu ke waktu
- Analisis performa penjualan per sales
- Analisis kontribusi penjualan berdasarkan wilayah

## Tools yang Digunakan
- Tableau
- Microsoft Excel

## Output
Dashboard interaktif yang menampilkan ringkasan KPI dan visualisasi penjualan
untuk mendukung pengambilan keputusan bisnis.
