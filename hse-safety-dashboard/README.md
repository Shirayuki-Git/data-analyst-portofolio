# HSE Safety Performance Dashboard

## Deskripsi
Dashboard ini digunakan untuk memonitor kinerja keselamatan kerja (HSE)
berdasarkan data insiden, skor keselamatan, dan jam pelatihan.

## KPI yang Dianalisis
- Total Incident
- Safety Score
- Training Hours

## Analisis Utama
- Tren insiden dari waktu ke waktu
- Hubungan antara training hours dan safety score
- Distribusi tingkat keparahan insiden (Low, Medium, High)
- Perbandingan safety score antar departemen

- Tableau
- Microsoft Excel
